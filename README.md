# Todo List REST API with Spring Boot

This is a simple RESTful API built with Spring Boot to manage tasks in a Todo List application.  
It supports basic CRUD operations, task filtering by status and priority, and due date queries.  

## Features

- Create, read, update, and delete tasks  
- Filter tasks by status and priority  
- Search tasks by name  
- Get tasks due before a certain date  
- Validation and error handling  
- API documentation with Swagger UI  
- Minimalistic frontend to interact with the API
